Dashboard Guide

Quick reference for dashboard functions

Sprint Header

Shows current date and days remaining in your active plan.

Always visible at top
Work days calculated (excludes weekends)
Goal Rings

Visual progress for each goal in your current plan. See at a glance what's on track.

Tap to increment progress
Ring fills as you complete
Red = behind, Yellow = at risk, Green = on track
Active Now

Your focus list. These are the tasks you're working on today.

Tap checkbox to complete
Color dot shows category/project
Edit Mode: bulk check + mark done
Backlog Boxes

Tasks grouped by category. Collapsed by default so you see the shape, not the noise.

Tap box to expand and see tasks
Tap task checkbox to activate immediately
Edit Mode: bulk select + activate
Number badge shows count
Edit Mode

Toggle for batch operations. Work fast, save once.

Tap [Edit] to enter
Check multiple tasks
"Done" button: complete all checked in Active
"Activate" button: move checked to Active
Tap [Save] to exit and apply
Color Key

Colors represent categories/projects. Scan and know what's what.
